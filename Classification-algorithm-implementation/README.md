# Classification-algorithm-implementation
Implementation and comparison of classification algorithms on the **Iris dataset** using Python and its scientific computing modules.

## Overview
- Performed **PCA** to reduce 4D data to 2D for visualization and better understanding of class separation
- Applied **train/test split** for model evaluation
- Compared models using **confusion matrix** and **F1-Score**

## Algorithms Implemented
| Model | Type | F1-Score |
|-------|------|----------|
| K-Nearest Neighbors (KNN) | Non-parametric | 0.924 |
| Random Forest (RF) | Non-parametric | 0.907 |
| Gaussian Naive Bayes (GNB) | Parametric | 0.907 |

**Best performing model: KNN with F1-Score of 0.924**

## Dataset
Iris Dataset — 4 features reduced to 2 dimensions using PCA
